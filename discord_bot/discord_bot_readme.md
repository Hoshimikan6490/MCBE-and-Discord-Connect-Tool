# 統合版マインクラフトとdiscordの相互通信ツール(アドオン)
## 概要 [2025/04/19　動作確認済み]
サーバー起動やログイン、ちゃっと送受信データをdiscordとBEマイクラサーバーの相互通信します。
- サーバーステータスの表示機能
- 接続ユーザー情報の表示(アドオンで実装)
- 接続アドレス、接続方法の提示
- ユーザー名の辞書登録機能
